# Maxma Sync Addon for CS-Cart

**Maxma Sync** — это аддон для CS-Cart, обеспечивающий интеграцию с платформой **Maxma/CloudLoyalty**. Аддон позволяет синхронизировать пользователей, заказы, корзины, бонусные программы и акции с внешней системой лояльности.

---

## Основные возможности

- Автоматическое создание и обновление профиля пользователя в Maxma.
- Синхронизация заказов и возвратов с Maxma.
- Расчет корзины с учетом бонусов и промо-акций.
- Применение внешних бонусов к корзине.
- Поддержка очередей обработки запросов для надежной синхронизации.
- Кэширование данных пользователей для ускорения работы.
- Логирование всех операций с возможностью анализа ошибок.
